Bootstrap Responsive Croogo Theme for Croogo CMS using Bootstrap by Twitter - Tested on Cake 1.3.15 + Croogo 1.3.5

For updates & screenshots visit http://nitishdhar.github.com/Bootstrap-Responsive-Croogo/

How to Use
===========
1. Just copy the unzipped folder in /app/views/themed/
2. Activate the Theme from admin panel

Note : This theme is still not for production use
